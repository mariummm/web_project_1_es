# Aprender a aprender
____
Este proyecto es una página web realizada siguiendo estrictamente las indicaciones del **curso de desarrollo web** como parte de una de las tareas que son requisito para aprobar dicho curso. 
Puede ser de utilidad para quienes estén incursionando en el mundo del desarrollo web. El código está ordenado según la metodología BEM para una mejor visualización y la anidación se realizó mediante un solo archivo CSS en el que se encuentran importadas todas las clases del código a través de la regla @import.  

## Características
Esta página web fue realizada mediante los lenguajes HTML5 y CSS los cuales se escribieron en el [Visual Estudio Code](https://code.visualstudio.com/) con la guía de la plataforma [PRACTICUM](https://practicum.com/es-mex/).
Se utilizó [w3schools.com](https://www.w3schools.com/) y [codebeautify](https://codebeautify.org/) para corregir y  ordenar el código HTML5.

**En el código de esta página encontrará:**

1.-Dimensiones en pixeles y porcentajes.

2.-Colores.

3.-Posicionamiento, opacidad.

4.-Márgenes, rellenos y bordes.

5.-Flexbox.

6.-Embebido de videos.

7.-Animación y transformación de imágenes.

8.-Tablas, solapamiento de imágenes.

9.-Iconos anidados.

10.-Código ordenado según la metodología BEM.
